# WIP: minihead

## Overview
linuxのheadコマンドをRustで置き換えた学習用のリポジトリです。
WIPです。

## Requirement
- WSL2
- cargo

## Usage
```
git clone ...
cd minihead
cargo build --release
./target/release/minihead 任意のファイルパス
```

## About debug
debugモードでのbuildは以下の手順で行えます。
```
git clone ...
cd minihead
cargo build
```
